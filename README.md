# Professional Full-Stack Portfolio

A high-performance, mobile-responsive portfolio website designed with a **mobile-first** approach. This project showcases real-time applications, technical expertise, and interactive UI elements.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3 (Flexbox/Grid)
*   **Interactivity:** Vanilla JavaScript (ES6+)
*   **Deployment:** Netlify / GitHub Pages

## Key Features
*   **Real-Time Projects:** Live links to functional web applications.
*   **Interactive UI:** Custom **hover effects**, scroll animations, and glassmorphism.
*   **Dynamic Skills Section:** Categorized technical stack and professional certifications.
*   **Direct Contact:** Integrated social media links and **instant email** functionality.
*   **Resume Download:** Direct access to my latest CV in PDF format.
*   **Optimized for Mobile:** Specifically tuned for high-resolution displays (like the Redmi series) with fluid 60fps animations.
